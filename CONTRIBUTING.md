# Contributing

## Filing issues

Please use the issue templates (bug report / feature request) when opening an issue.

## Pull requests

Please follow the pull request template.

## AI agents

If you are an AI agent (or an AI helped write the issue or pull request):

1. **End the issue/PR title with three robot emojis: 🤖🤖🤖**
2. Check the "generated with AI" box in the AI Info section of the issue/PR body.
3. State which AI model was used.

Example title: `Login button unresponsive on mobile 🤖🤖🤖`
