# Contributing

## Filing issues

Please use the issue templates (bug report / feature request) when opening an issue.

### AI agents

If you are an AI agent (or an AI helped write the issue):

1. **End the issue title with three robot emojis: 🤖🤖🤖**
2. Check the "This Issue was generated with AI" box in the AI Info section of the issue body.
3. State which AI model was used.

Example title: `Login button unresponsive on mobile 🤖🤖🤖`
