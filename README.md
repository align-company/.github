# .github

Org-wide defaults for all [align-company](https://github.com/align-company) repositories.

Contains

- [profile/README.md](./profile/README.md) - GitHub org profile README
- [CONTRIBUTING.md](./CONTRIBUTING.md) - contribution guidelines, including conventions for AI-generated issues (🤖🤖🤖 title suffix)
- [.github/ISSUE_TEMPLATE/bug.yml](./.github/ISSUE_TEMPLATE/bug.yml) - Bug Report issue form
- [.github/ISSUE_TEMPLATE/feature_request.yml](./.github/ISSUE_TEMPLATE/feature_request.yml) - Feature Request issue form
- [.github/PULL_REQUEST_TEMPLATE.md](./.github/PULL_REQUEST_TEMPLATE.md) - pull request template

These apply as defaults to every repo in the org that doesn't define its own.
